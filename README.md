# Space Exploration App

This is a simple full-stack project that fetches and displays space-related data using NASA APIs. It uses Node.js and Express.js for the backend and React for the frontend.

## Features

* Fetches data from NASA APIs
* Basic REST API using Express.js
* Displays data using a React frontend

## Tech Stack

* Node.js
* Express.js
* React.js
* NASA API
