Space Exploration App:-

This is a simple full-stack project that fetches and displays space-related data using NASA APIs. It uses Node.js and Express.js for the backend and React for the frontend.

Features:-
1.Fetches data from NASA APIs
2.Basic REST API using Express.js
3.Basic REST API using Express.js
4.Displays data using a React frontend
5.Tech Stack
6.Node.js
7.Express.js
8.React.js
9.NASA API
